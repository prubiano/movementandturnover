# Movement and Turnover

This project aims to track and analyze movement and turnover metrics.

## Features
- Track employee movement
- Analyze turnover rates
- Generate reports

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/prubiano/movementandturnover.git
   ```
2. Navigate to the project directory:
   ```
   cd movementandturnover
   ```

## Usage
- Follow the documentation for setting up the environment and running analyses.
